Código Alfabético

Esse método transforma letras em números, seguindo a ordem do alfabeto. A letra A vira 1, B vira 2, C vira 3... até o Z, que vira 26. É uma forma simples e didática de representar palavras numericamente.

Exemplo:

Amor → 1 13 15 18

Base64

Base64 é uma forma de converter dados (como textos ou arquivos) em uma sequência segura de caracteres. Muito usada para transmitir dados via internet, pois transforma tudo em letras e números legíveis. Não é uma criptografia forte, mas sim uma codificação.

Exemplo:

Decode → ZGVjb2Rl

Cifra de César

Criada por Júlio César, essa técnica troca cada letra por outra, algumas posições adiante no alfabeto. Se o deslocamento for 3, o A vira D, o B vira E e assim por diante. É uma das criptografias mais antigas.

Exemplo:

Rap → uds

ROT13

ROT13 é um tipo especial de Cifra de César com deslocamento fixo de 13 letras. Se aplicar ROT13 duas vezes, volta ao texto original. Muito usada em fóruns e mensagens onde o conteúdo deve estar escondido, mas fácil de decifrar.

Exemplo:

Segredo → frterqb

Hexadecimal

O sistema hexadecimal representa dados usando números de 0 a 9 e letras de A a F. Cada caractere é traduzido com base na tabela ASCII. Muito usado em programação, redes e segurança digital.

Exemplo:

Web → 77 65 62

Código Morse

Código Morse usa sinais curtos (.) e longos (–) para representar letras e números. Foi muito usado em rádios e telégrafos. Mesmo simples, ainda é uma forma eficiente de comunicação em situações extremas.

Exemplo:

SOS → ... --- ...
